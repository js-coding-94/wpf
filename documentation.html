Getting familiar with Foundation

At first I had to go visit the Foundation-Website (picture has a link) and download Foundation 5. There are many sites where you can look up the documentation of the included framework components.

So did I and started working on my own website including several components, such as "Magellan", "Grid", "Tabs" etc. Foundation offers a lot of useful tools to customize the look of your website, plus: It's full responsive! Moreover I had to change some CSS-references in order to get a specific look.

Setting up SCSS

Using Foundation offers a great possibility to use the preprocessing language SCSS (Syntically Cascading Style Sheets). With the use of this language, you are able to create variables, nested elements, mixins and many more possibilities to make your CSS-styles more organized.

To use SCSS, I also had to download the precompiler Koala, which translates SCSS-code to customized CSS automatically and this method worked very well.

HTML- & SCSS Coding with Framework Components

For my HTML and SCSS coding I used the texteditor programm Sublime, which basically offers Syntax Highlighting and an autocomplete function for several web-based languages.

Personally, I used both Foundation classes and my own classes to customize the look of my website. In the end I had problems with Sublime due to file path problems. Therefore I switched to another texteditor programm named Brackets, which also worked out quite well.

Transfering project to GitHub

After the coding was finished the next task was to transfer this project to GitHub. First, I had to create an account in order to get access to the services.

The next step was create a profile and a new project.
